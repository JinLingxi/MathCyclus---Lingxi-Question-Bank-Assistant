# 📚 高中数学题库管理与组卷系统

这是一个基于 Python (Streamlit) 和 LaTeX 的自动化高中数学题库管理与智能组卷系统。专为中学数学教师及教研团队设计，旨在彻底解决海量 LaTeX 题目的碎片化管理与高效排版问题。

## ✨ 核心功能

- **📝 题库管理与自动化归档**：一键批量读取题目源文件，利用正则解析题目标签，并按「板块/年份」智能分类归档。
- **🖨️ 试卷排版工作台**：支持可视化选题、上下调整题目顺序、插入大题/小节标签，右侧提供 LaTeX 题目的实时排版预览（Markdown 渲染）。
- **🤖 AI 辅助服务**：集成阿里云百炼大模型能力，支持上传题目截图进行高精度 OCR 识别并转换为 LaTeX 代码，同时支持基于特定模板的 AI 辅助智能组卷。
- **🎨 TikZ 绘图深度整合**：内置本地系统（`xelatex` + `PyMuPDF`）的 TikZ 代码实时编译服务，可将复杂的几何图形代码实时转为 PNG 图片预览并分离存放，彻底告别盲写代码。

## 🛠️ 技术栈

- **前端界面与交互**：[Streamlit](https://streamlit.io/) 响应式框架
- **排版与编译引擎**：**$\LaTeX$** (核心依赖 `xelatex` 编译器)
- **底层脚本与处理**：Python (`re` 文本处理, `fitz` PDF图像解析, `subprocess` 进程调用)
- **大语言模型支持**：阿里云百炼 (Qwen 模型系列)

## 🚀 快速启动

1. 确保您的电脑已安装 **Python 3.8+** 以及以下依赖库：

   ```bash
   pip install streamlit pymupdf python-dotenv
   ```
2. 确保本地已安装完整的 **$\LaTeX$ 编译环境**（例如 TeX Live），且 `xelatex` 命令已加入系统环境变量中。
3. 在项目根目录的 `.env` 文件中配置您的 AI 模型密钥（如阿里云 API KEY）。
4. 双击运行 `启动程序.bat`，或者在终端执行：

   ```bash
   python 启动程序.py
   ```

   即可自动在浏览器中打开工作台。

## 📁 核心目录结构

```text
├── chapters/              # 存放按学科板块和年份分类的 LaTeX 题库源文件
├── utils/                 # 系统核心工具库 (配置、文件读写、TikZ 渲染、LaTeX 解析)
├── question_bank_app.py   # Streamlit 主程序入口
├── 启动程序.py             # 守护进程与服务启动脚本
└── log.csv                # 批量题目处理与归档日志
```
